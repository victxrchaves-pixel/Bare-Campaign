# Brief — Teaser UI Pillar

## Goal
Show Bare's interface to generate product desire without revealing real users.
Not a demo — a visual seduction. The empty architecture communicates more than a filled screen.

---

## Screens to show (by impact order)

### Screen 1 — Empty search
The creator search view with visible filters: discipline, palette, technique, atmosphere.
No results loaded. Only the filter architecture visible.
Overlaid copy: "FILTER BY PALETTE. BY TECHNIQUE. BY ATMOSPHERE."

### Screen 2 — Blank profile
An empty creator profile, clean, with visible structure.
Projects as empty blocks — the portfolio architecture without content.
Overlaid copy: "YOUR WORK. ONE PLACE."

### Screen 3 — The archive
The curated inspiration module. Reference grid with no images loaded.
Only the structure and metadata of one reference entry (palette, technique, context).
Overlaid copy: "INSPIRATION WITH CONTEXT. NOT ACCUMULATION."

### Screen 4 — The access
The review process screen. The project submission form.
Overlaid copy: "YOU SUBMIT A PROJECT. WE REVIEW IT. YOU'RE IN OR YOU'RE NOT."

---

## Visual treatment of screens

- All screens in dark mode
- Convert to black and white — no color in any published screen
- Overlay copy in Switzer SemiBold, #FAFAFA, uppercase, left-aligned
- Overlay copy sits in bottom third of the image, with 8% margin from edges
- Add "bare" signature in Geist Light, #FAFAFA, bottom left corner, wide tracking
- Background of overlaid text: no box, no background — text sits directly over the screen

---

## Formats to produce per screen

| Format | Dimensions | Use |
|---|---|---|
| IG Feed square | 1080×1080px | Feed post |
| IG/TK vertical | 1080×1920px | Stories and TikTok |

Each screen produces 2 assets: 1 feed + 1 vertical.
Total assets from this brief: 8 files (4 screens × 2 formats).

---

## TikTok adaptation

For vertical TK format, the screen occupies the top 60% of the frame.
Copy animates in from the bottom over 3–4 seconds.
"bare" signature fades in last and holds for 2 seconds.
No voice over — text only, silence or minimal ambient audio.

## Publishing cadence

| Month | Cadence | Notes |
|---|---|---|
| Month 1 | 1 Teaser UI per week (weeks 1–4) | No CTA — identity only |
| Month 2 | 0 — pause | Explicativo and FOMO take over |
| Month 3 | 2 Teaser UI total (weeks 9 and 11) | Pair with urgency copy from Block B |

Never publish two Teaser UI pieces on the same day.
In Month 3, pair with Block B copy — e.g. Screen 1 + Copy B1, Screen 4 + Copy B4.
