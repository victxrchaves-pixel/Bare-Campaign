- No search results loaded — filters visible, results area empty
- Convert to black and white, high contrast
- UI sits flush to top and sides of the image

**Typography overlay:**
- "FILTER BY PALETTE. / BY TECHNIQUE. / BY ATMOSPHERE." → Switzer SemiBold, ~52pt, #FAFAFA, left-aligned
- 8% margin from left and bottom edges
- Separator line: 1px solid #FAFAFA, 40px wide, above copy block
- "bare" signature → Geist Light, 16pt, #FAFAFA, letter-spacing 0.3em, bottom-left below copy

**Notes for designer:**
- The UI screen must be legible enough to feel real but not so detailed it distracts from the copy
- If the UI isn't ready, use a wireframe-fidelity version — the architecture matters, not the polish
