# Post 05 — Waitlist · Single Image (FOMO)
**Phase:** Waitlist · Week 5–8 (repeat each batch opening)  
**Format:** 1080×1080px  
**Type:** Single image — batch opening, FOMO

---

## Visual

**Background:** #FAFAFA  
**Layout:** Left-aligned, tension between large number and small text

```
[top — empty breath]

30
SPOTS
THIS WEEK.

─────────────

BATCH [N] · CLOSES SUNDAY.

bare · waitlist in bio
```

**Typography:**
- "30" → Switzer SemiBold, ~280pt, #101010, left-aligned, 8% margin — dominates the frame
- "SPOTS / THIS WEEK." → Switzer Regular, ~60pt, #101010, left-aligned, tight to "30"
- Separator line → 1px solid #101010, full width minus margins
- "BATCH [N] · CLOSES SUNDAY." → Geist Regular, ~22pt, #101010, letter-spacing 0.15em, uppercase
- "bare · waitlist in bio" → Geist Light, ~18pt, #101010, letter-spacing 0.25em

**Variable elements (update each week):**
- Replace [N] with the batch number (1, 2, 3, 4)
- Replace "CLOSES SUNDAY" with the actual closing day if different

**Notes for designer:**
- The "30" must feel oversized — it's the whole message
- Keep the bottom section minimal, the number does the work
